원티드랩 데이터팀에서 사용하는 쿼리 생성 AI. 데이터 요청 처리 자동화와 데이터팀의 업무 부담 경감을 위해 개발되었으며 슬랙 채널을 통해 데이터 요청 시 적절한 SQL 쿼리를 생성. 원티드랩의 데이터 구조를 이해하고 효율적으로 쿼리를 제공하는 특징이 있고 마케터, PO, 데이터 분석가 등이 쿼리 생성 및 수정에 활용 중이며 SQL 진입 장벽 감소
